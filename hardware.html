Motherboard-

The motherboard is one of the most crucial element of a computer, it's the main circuit board of a computer. It permits the hardware in the computer to communicate with each other. It can expand by plugging other circuit boards into it.

CPU (Central Processing Unit)-

The CPU is a component that controls the other computer components. By sending out signals, the CPU gives the other components instructions on what to perform, by turning data input into information output. The CPU determines the capability of the system. In recent years the top performance cpus were made by these two manufacturers, AMD and Intel. There are many more manufacturers that have a trusted name, but AMD and Intel's cpus tend to have higher customer satisfaction and reviews. Extra Info- it's speed is measured in MHz and GHz.









RAM (Random Access Memory)-

RAM is temporary storage for data. It's accessed when data is currently being in use. The processor in the computer quickly accesses it every time you use your computer. When your computer is not in use, all of the RAM is lost (all of it's data is lost). When the RAM is full of data, your computer will start working slower due to the hard disk overlaying information to help the computer keep up. Due to your computer constantly using RAM, it was made to be easily directly accessed making it easier to retrieve recent data.








Hard Drive-

A Hard Drive is a storage area. The Operating System (OS) is stored in your hard drive. It's the long term memory of your computer which stores pretty much everything that's kept in your computer (files, programs, preference, OS, etc.).








Keyboard/Mouse/Monitor

A keyboard is a set of keys that input information into a computer to operate it. A mouse moves the cursor on a computer screen. Lastly a monitor display screen that displays the graphics information generated by your computer. Both the keyboard and the mouse input information into the computer, while the monitor outputs information to the user of the computer.
